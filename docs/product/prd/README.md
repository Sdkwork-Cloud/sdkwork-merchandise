# Product PRD Directory

This directory owns the product Canon for the repository.

## Fixed Entry

- [PRD.md](PRD.md) is the required entry document. Keep summary, status, and links here.

## Splitting Rules

- Split large PRD content into sibling shards named `PRD-<kebab-topic>.md`.
- Every shard must be linked from `PRD.md`.
- Do not create competing product roots such as `docs/product/PRD.md`; that path is redirect-only.

See `../../../../sdkwork-specs/DOCUMENTATION_SPEC.md` section 2.2.
